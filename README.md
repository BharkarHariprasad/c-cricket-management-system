# 🏏 Cricket Management System (C Programming)

## Overview

Cricket Management System is a console-based application developed using **C programming**. The system manages basic cricket player records including runs, wickets, matches played, and jersey numbers.

The project demonstrates fundamental programming concepts such as **structures, arrays, sorting algorithms, searching techniques, and modular function design**.

This project was developed during the early stage of my programming journey to practice structured programming and logical problem solving.

---

## Features

- Display all players
- Add new players
- Remove player records
- Update player details
- Search players by:
  - Jersey number
  - Player name
- Sort players by:
  - Runs
  - Wickets
